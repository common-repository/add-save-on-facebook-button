The plugin allows you to add the new Save on Facebook button to your articles.

== Description ==

The plugin allows you to insret the Save on Facebook button to your articles. You can place it at the bottom ro
at the top of the articles. You can choose if you want it to display only on blog posts or on pages as well.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Save on Facebook settings to customize the position of the button.
1. Facebook's JavaScript SDK is being loaded by default. If you have other plugins inserting the the SDK, uncheck the Add SDK option.
1. Enjoy the new button!


== Frequently Asked Questions ==

= All my Facebook features stopped working. What's wrong? =

Most probably you have another plugin installed that also includes the Facebook JavaScript SDK. To fix this, go to Save on Facebook settings and uncheck the "Add Sdk" option.
